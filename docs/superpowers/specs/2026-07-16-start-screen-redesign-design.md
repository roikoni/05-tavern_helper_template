# StartScreen.vue 水墨封面改版设计

## 背景

`src/开场/components/StartScreen.vue` 当前封面 UI 过于复杂：96 粒飘雪动画喧宾夺主、标题「黑色修士」套着厚重的半透明墨匣背景框、「踏入此道」按钮带有四角墨点和厚重渐变。用户有一张黑白水墨画底图很满意，但 UI 破坏整体氛围，需要简化与精炼。

## 目标

将封面 UI 从「装饰堆砌」转向「克制留白」，让水墨画底图成为绝对主角，UI 元素仅作最低限度的点缀和引导。

## 设计要点

### 1. 墨雾流动（替代飘雪）

- 用 5 个大小不一的墨团通过 CSS animation 从左向右缓缓漂移
- 三层视差：速度 22s / 26s / 30s / 32s / 28s，各有不同延迟
- 透明度 6%–12%，非常克制不抢戏
- 滤镜 `blur(60px)` 柔化边缘
- 垂直分布覆盖画面不同高度

### 2. 标题「黑色修士」

- **去掉**当前厚重的半透明墨匣背景框（`::before` 伪元素的 backdrop-filter + 边框）
- **保留**书法字体（`$font-brush`）和冷玉渐变文字效果
- **弱化** `::after` 底部阴影斜线
- **新增**精炼的 text-shadow，让文字从水墨画底图中浮现而非嵌在框里
- **新增**标题下方一条淡墨横线装饰，中央点缀小墨点

### 3. 「踏入此道」按钮

- **去掉**四角墨点（`text::before` 的 radial-gradient）和厚重渐变背景
- **改为**透明底 + 1px 淡色边框，文字 letter-spacing 加大
- **悬浮时**：边框提亮 + 极淡底部墨晕染层 + 两侧太极符变血色微光
- **去掉** `box-shadow` 的厚重 inset 效果

### 4. 暗角遮罩

- 增强径向暗角，中心和边缘对比更强烈
- 确保文字在任何底图上都有足够可读性

### 5. 退场动画

- 保留现有的水墨晕开退场（`ink-bloom`），与新的墨雾风格统一
- `ink-out` 类仍绑定 `filter: blur(2px) brightness(0.5)`

## 技术约束

- 继续使用 `<style scoped lang="scss">`
- 继续使用 `@use './theme' as *;` 引入主题变量
- 保持 `aspect-ratio: 4 / 3`（桌面）和 `3 / 4`（移动端）
- 保持现有事件发射逻辑（`emit('enter', '普通')`）不变

## 验收标准

1. 封面不再显示飘雪粒子，取而代之的是淡淡墨雾流动
2. 标题无背景框，仅靠文字渐变和 text-shadow 浮现于水墨画之上
3. 按钮透明底 + 细边框，悬浮时有微妙的血色光晕
4. 暗角遮罩足够强，文字在任何底图区域都可读
5. 点击「踏入此道」后仍触发水墨晕开退场动画
6. 移动端比例和字号适配正常
